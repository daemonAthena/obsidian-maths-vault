And is a logical binary operation which takes two values of the set of $\mathbb{Z} / 2$ and applies the "conjunction" function which only returns 1 only if both operands are 1 and otherwise returns 0.

The operation is noted as $\land$ throughout my work. `\land` in LaTeX.

| Arg 1 | Arg 2 | Return |
| :---: | :---: | :----: |
|  $1$  |  $1$  |  $1$   |
|  $1$  |  $0$  |  $0$   |
|  $0$  |  $1$  |  $0$   |
|  $0$  |  $0$  |  $0$   |
