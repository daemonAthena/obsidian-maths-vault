# Definition
A topology, on a set $S$, $\mathcal{T}$ is a collection defined as follows:
- Must contain $S$ and $\emptyset$
- Must be closed under finite intersections
- Must be closed under unions

A set which is a member of the topology is called an **open set**
# Topological Space
A topological space is a set and Topology $(S,\mathcal{T})$

# Open Cover
An open cover, $O$ ,of a set $X \subset (S,\mathcal{T})$ is a collection of open sets whose union contains X as a subset.

In other words, $O$ is an open cover of $X$ if
$X \subset \bigcup_{A\in O} A$
where each $A$ is an open set

## Subcover
A subcover is simply a subset of another cover which is still a cover.
so in this case $U \subset O$ where all properties of $O$ still hold.

# Compactness
A topological space, $\mathcal{T}$, is compact if
every open cover of $S$ has a finite subcover of $S$

For all $O \subset \mathcal{T}$ where $S \subset \bigcup_{A\in O}A$, There exists finite $U \subset O$ such that $S \subset \bigcup_{A\in U}A$
